Directions

There are 3 main operations that can be performed on this page:

  1. Select the data you want to visualize from the dropdown menus. There are 2 options, so that you can compare the data.
  2. Hover over the bars in the histograms, scatterplot points, or the counties in the choropleth maps to see the data values.
  3. Click and drag (to brush) to select a range of values in the histograms and scatterplot. This will highlight the counties with that data in all visualizations.
